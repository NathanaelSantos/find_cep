# HTTP Requests in Flutter
## A composable, Future-based library for making HTTP requests.

## Step 1 - add library
### With Dart:
 `dart pub add http`
 
### With Flutter:
 `flutter pub add http`
 
### Now in your Dart code, you can use:

`import 'package:http/http.dart';`
